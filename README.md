# Проект avitoQA

Две части задания:
- `Task1/` — решение первой части (см. `Task1/Task1.md`).
- `Task_2/` — E2E тесты таск-трекера (Playwright). Инструкции в `Task_2/README.md`, тест-кейсы в `Task_2/TESTCASES.md`, баг-репорты в `Task_2/BUGS.md`.

Как запустить тесты (Task_2):
1. `cd Task_2`
2. `npm install`
3. `npm test`

Основной сайт для проверки: `https://avito-tech-internship-psi.vercel.app/`.
