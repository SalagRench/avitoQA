## Известные проблемы (баг-репорты)

### BUG-001: 404 при прямом заходе на `/issues`
- **Приоритет:** High  
- **Шаги:** открыть `https://avito-tech-internship-psi.vercel.app/issues` напрямую.  
- **ОР:** страница со списком задач.  
- **ФР:** Vercel 404.  
- **Комментарий:** обход — открыть корень `/`, происходит редирект на рабочую `/issues`.

### BUG-002: Поле «Статус» обязательное, но всегда Backlog и disabled
- **Приоритет:** Medium  
- **Шаги:** открыть модалку создания задачи.  
- **ОР:** статус можно выбрать (или поле не отмечено как обязательное).  
- **ФР:** поле помечено `*`, но доступно только значение Backlog (disabled).  
- **Комментарий:** пользователь не может выбрать другой статус при создании.

### BUG-003: Редактирование задачи не сохраняет изменения
- **Приоритет:** High  
- **Шаги:** открыть карточку задачи → изменить название → сохранить → закрыть.  
- **ОР:** новое название отображается в списке.  
- **ФР:** список остаётся со старым названием.  
- **Комментарий:** изменения не применяются.

### BUG-004: Фильтр по статусу `Done` показывает задачи других статусов
- **Приоритет:** High  
- **Шаги:** на списке задач открыть фильтр «Статус» → выбрать `Done`.  
- **ОР:** остаются только задачи со статусом `Done`.  
- **ФР:** отображаются задачи со всеми статусами.  
- **Комментарий:** фильтр статуса работает некорректно.
